# Coffee Chats Networking App

Coffee Chats is a **mobile-first React web application** designed to help users discover mentorship and networking opportunities through guided onboarding and preference-based matching.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Supabase Auth + Postgres
- Vercel

## Features

- Multi-step onboarding flow
- Preference-based discovery feed
- Match detail experience
- Coffee chat request workflow
- Messaging interface
- Scheduling interface

## Project Status

Active development — MVP in progress.
